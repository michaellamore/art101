Challenge

The challenge for this lab was to use loops in order to iterate using a max number, and be able to print all the numbers onto the HTML. We're also experimenting with iterators, string manipulation, and being able to output things onto the document through javascript.

Problems

The lab itself was simple enough to follow. I've already experimented with for loops before. I also didn't have much time to do this lab because of other time commitments I had to do, so the code is not pretty. Very hard to read, I would have to work on that for future labs.
